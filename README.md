## How to use

1. `git https://github.com/tomanagle/typescript-starter.git my-app`
2. `cd my-app`
3. `yarn install`
4. `yarn dev`

## Includes

-   [x] TypeScript
-   [x] Nodemon
-   [x] Jest
-   [x] ESLint
-   [x] Prettier

## Scripts

### Start the server in development mode

`yarn dev`

### Run Jest

`yarn test`

### Run the linter

`yarn lint`

### Run the linter anf fix errors

`yarn lint:fix`

### Build into the dist directory

`yarn build`
